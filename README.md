# 宫颈细胞学AI辅助诊断系统（鸿蒙版）

本项目是一个基于鸿蒙系统开发的宫颈细胞学AI辅助诊断应用，利用华为ModelArts平台的深度学习能力，实现对宫颈液基细胞学（LBC）图像的自动分析和辅助诊断。

## 功能特点

- **图像上传与显示**：支持从设备相册选择宫颈细胞学图像
- **AI辅助诊断**：调用ModelArts云端API进行细胞分类诊断
- **诊断结果展示**：直观显示检测到的细胞类型、置信度及诊断建议
- **系统介绍**：提供系统技术特点和诊断指标信息

## 技术架构

- **前端**：鸿蒙系统ArkUI框架
- **后端**：华为ModelArts云端推理服务
- **通信**：HTTP请求实现前后端交互

## 系统要求

- 鸿蒙系统3.0及以上版本
- 支持的设备类型：手机、平板
- 需要网络连接权限
- 需要媒体文件读取权限

## 使用说明

1. **启动应用**：安装并打开应用
2. **授予权限**：允许应用访问网络和媒体文件
3. **选择图像**：点击"选择宫颈液基细胞学图像"按钮，从相册中选择图像
4. **开始诊断**：点击"开始AI诊断"按钮，等待分析结果
5. **查看结果**：诊断完成后，查看细胞类型、置信度及诊断建议

## ModelArts API配置

使用前需要在Index.ets文件中配置您的ModelArts推理服务信息：

```javascript
// 模型API配置
const MODEL_API_URL = 'https://your-modelarts-inference-endpoint';
const API_KEY = 'your-api-key-here';
```

## 诊断类型

系统能够识别以下四种细胞类型：
1. 正常细胞
2. ASC-US（非典型鳞状细胞，意义不明确）
3. LSIL（低级别鳞状上皮内病变）
4. HSIL（高级别鳞状上皮内病变）

## 免责声明

本系统仅提供辅助诊断建议，不能替代专业医生的诊断。最终诊断结果请以临床医生的判断为准。

### 依赖说明

本应用主要使用鸿蒙系统内置的API，包括：
- @ohos.multimedia.image：图像处理
- @ohos.net.http：网络请求
- @ohos.file.picker：文件选择
- @ohos.permission：权限管理

## 注意事项

1. 首次使用时需要授予必要的权限
2. 诊断过程需要稳定的网络连接
